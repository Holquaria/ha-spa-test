## Getting Started

To run the application locally, please clone into your local environment and run

```bash
npm install
# or
yarn install
# or
pnpm install
# or
bun install
```

Then run

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
